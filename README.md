<p>Veja as principais funcionalidade do app</p>

<ul>

  <li>Tela de cadastro</li>
  <li>Pesquisar amigos</li>
  <li>Alterar dados do perfil</li>

  <li>Feed de Postagens</li>

  <li>Listar Comentários</li>
  <li>Tela de Perfil</li>
